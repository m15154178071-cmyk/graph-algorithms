# 最小环基算法（Minimum Cycle Basis, MCB）

本仓库提供了无向图**最小环基（MCB）**问题的 Python 实现，包含两个版本，适用于不同的性能需求和学习场景。

## 文件说明

### 1. `solution_basic.py`（基础版）
- **简介**：注重算法逻辑的基础实现。
- **特点**：
  - 标准 GF(2) 高斯消元。
  - 基本环枚举（C3/C4）。
  - 结构清晰、扁平，适合学习核心思想。
- **适用场景**：适合小规模图和教学用途。

### 2. `solution_enhanced.py`（增强版/工程版）
- **简介**：面向性能优化的工程实现。
- **特点**：
  - **O(1) 树路径异或**：通过预处理根到节点的异或掩码，加速基本环计算。
  - **C3-C5 枚举**：显式枚举长度不超过 5 的环。
  - **引导式 DFS**：利用结构信息（H-structure）引导深度优先搜索，处理更长的环（C6+）。
  - **健壮性**：严格的“诱导环”检查与内存管理。
- **适用场景**：推荐用于竞赛或处理中大型图（1万~50万边）。

## 运行方法

```bash
python solution_enhanced.py input.txt output.txt
```

## 环境要求
- Python 3.8 及以上
- 无需额外依赖（仅用标准库）
